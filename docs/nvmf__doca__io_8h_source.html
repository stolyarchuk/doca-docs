218 /*
219  * Create NVMf DOCA IO
220  *
221  * Creates an IO, which can be used to receive DBs on CQ and associated SQs, read SQEs from Host, write CQEs to Host
222  * and raise MSI-X.

226  * @return: DOCA_SUCCESS on success and DOCA_ERROR otherwise
