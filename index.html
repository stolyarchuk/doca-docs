NVIDIA DOCA SDK Documentation

Introduction

Welcome to the NVIDIA DOCA (Data Center on a Chip) SDK documentation. DOCA is a comprehensive software framework that enables developers to harness the power of NVIDIA BlueField DPUs (Data Processing Units) and SuperNICs for accelerated computing in data centers.

DOCA Components

The DOCA SDK includes the following key components:

Network Processing

  • DOCA Flow: Advanced packet processing and flow management
  • DOCA DMA: High-performance direct memory access operations
  • DOCA RDMA: Remote direct memory access for low-latency networking
  • DOCA Ethernet: Ethernet queue management and data path acceleration

Security and Encryption

  • DOCA AES-GCM: Hardware-accelerated AES-GCM encryption/decryption
  • DOCA SHA: Secure hash algorithm implementations
  • DOCA App Shield: Application and system security monitoring

Storage and Compression

  • DOCA Compress: Hardware-accelerated compression and decompression
  • DOCA Erasure Coding: Advanced data protection and recovery

Acceleration and Offloading

  • DOCA DPA: Data Path Acceleration for custom packet processing
  • DOCA GPU NetIO: GPU-accelerated network I/O operations
  • DOCA PCC: Precision Clock Control for timing-critical applications

Monitoring and Telemetry

  • DOCA Telemetry: Comprehensive system and application monitoring
  • DOCA Telemetry Exporter: Data export capabilities for analytics

Getting Started

To begin developing with DOCA:

  1. Explore the API Reference: Navigate through the modules to understand available functions and data structures
  2. Review Sample Applications: Check the applications and samples directories for working examples
  3. Read the Headers: Examine the include files for detailed API documentation
  4. Build and Test: Use the provided build system to compile and test your applications

Development Support

  • Headers: Complete API definitions in the include/ directory
  • Samples: Working examples in the samples/ directory
  • Applications: Full applications in the applications/ directory
  • Tools: Development and debugging tools in the tools/ directory

Target Architecture

DOCA applications are designed to run on:

  • NVIDIA BlueField DPUs: ARM64-based data processing units
  • Host Systems: x86_64 systems with NVIDIA networking hardware
  • Development Environment: Cross-platform development with container support
